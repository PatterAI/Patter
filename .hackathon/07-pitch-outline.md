## Q&A Prep (5 minutes)

### "How is this different from Khan Academy?"
Khan Academy is a content library — the same videos for every student. LEO generates custom content aligned to each teacher's specific curriculum and adapts difficulty per student in real-time using Bayesian mastery tracking. Teachers bring their syllabus; LEO builds the learning experience around it.

### "What's your moat?"
Three reinforcing loops:
1. **Adaptive engine** — more student interactions = better mastery predictions = better learning outcomes
2. **Content generation** — teacher-created modules become training signal for more effective generation
3. **Analytics flywheel** — better data = better insights = more teacher trust = more adoption

The combination of all three in one platform is the moat. Competitors would need to build all three simultaneously.

### "How do you handle hallucinations in AI-generated content?"
- **Teacher review gate** — all AI-generated content requires teacher approval before going live
- **Structured prompts** — content generation uses educational taxonomy frameworks (Bloom's, Webb's DOK) as guardrails
- **Student feedback loops** — students can flag confusing or incorrect content, which triggers re-review
- **Source attribution** — generated notes include citations and references for teacher verification

### "What's your go-to-market strategy?"
Bottom-up, product-led growth:
1. Individual teachers discover LEO (content marketing, teacher communities, conference demos)
2. Free tier proves value with measurable analytics (mastery improvements, time saved)
3. Teachers share with colleagues — organic spread within a school
4. School admin sees adoption + outcome data -> enterprise conversation
5. District deals driven by demonstrated ROI across multiple schools

### "How do you measure learning outcomes?"
- **Mastery scores** tracked longitudinally per student per subtopic
- **A/B testing** of content effectiveness (which content types drive faster mastery for which topics)
- **Pre/post assessments** — standardized comparisons before and after LEO usage
- **Teacher satisfaction** — time saved on content creation and grading, measured via in-app analytics
- **Engagement metrics** — session duration, content completion rates, voluntary return rates

### "What about data privacy and COPPA/FERPA compliance?"
- All student data is encrypted at rest and in transit
- Multi-tenant architecture ensures organization-level data isolation
- No student PII is sent to AI models — content generation uses topic data only
- FERPA-compliant data handling designed into the architecture from day one
- Enterprise tier includes dedicated data processing agreements

### "Why should we invest now?"
- AI capabilities (GPT-4, Gemini) have reached the quality threshold for reliable educational content generation
- EdTech spending is accelerating post-pandemic
- First-mover advantage in the adaptive + generative intersection
- Working prototype demonstrates technical feasibility — this is a go-to-market funding round, not R&D
