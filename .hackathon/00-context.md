## Patter Project
- **What Patter is:** Open-source platform that connects AI agents to phone calls. Python + TypeScript SDKs. Local mode (Twilio/Telnyx + STT/TTS) and cloud mode.
- **Repo:** /Users/francescorosciano/docs/patter/Patter
- **Key capability:** 10 lines of code to give an AI agent a phone number

## Francesco's Initial Ideas

### FastShot Track Idea (strongest lead)
A phone-based interface for Claude Code sessions:
- **Option A:** You call a number and an AI voice agent tells you how your Claude Code coding session is going (progress, blockers, what it just shipped)
- **Option B:** Claude Code proactively calls YOU when something important happens (build failed, feature shipped, needs a decision)

This leverages Patter's core capability (AI agents on phone calls) and connects it to the developer tools space.
