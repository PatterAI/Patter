## Python

### 1. Install

```bash
pip install patter[local]
```

The `[local]` extra includes Twilio, FastAPI, and uvicorn for self-hosted mode.

### 2. Set up credentials

You need a Twilio or Telnyx account and an OpenAI API key.

```bash
export TWILIO_ACCOUNT_SID=AC...
export TWILIO_AUTH_TOKEN=...
export OPENAI_API_KEY=sk-...
```

### 3. Write your agent

```python main.py
import os
from patter import Patter

phone = Patter(
    twilio_sid=os.environ["TWILIO_ACCOUNT_SID"],
    twilio_token=os.environ["TWILIO_AUTH_TOKEN"],
    openai_key=os.environ["OPENAI_API_KEY"],
    phone_number="+15551234567",  # Your Twilio number
    webhook_url="abc123.ngrok.io", # Your public URL (no https://)
)

agent = phone.agent(
    system_prompt="You are a friendly assistant. Keep responses brief.",
)

await phone.serve(agent, port=8000)
```

### 4. Expose your server

```bash
ngrok http 8000
```

### 5. Call your number

Dial the phone number you configured. Your AI agent will answer.

## TypeScript

### 1. Install

```bash
npm install getpatter
```

### 2. Write your agent

```typescript index.ts
import { Patter } from "getpatter";

const phone = new Patter({
  twilioSid: process.env.TWILIO_ACCOUNT_SID,
  twilioToken: process.env.TWILIO_AUTH_TOKEN,
  openaiKey: process.env.OPENAI_API_KEY,
  phoneNumber: "+15551234567",
  webhookUrl: "abc123.ngrok.io",
});

await phone.serve({
  agent: {
    systemPrompt: "You are a friendly assistant. Keep responses brief.",
  },
});
```

### 3. Expose and call

```bash
ngrok http 8000
```

Dial your number — your agent is live.

---

## What just happened?

1. **Patter started an embedded HTTP/WebSocket server** on port 8000
2. **Twilio was configured** to send incoming calls to your ngrok URL
3. **When someone calls**, Twilio opens a media stream WebSocket to your server
4. **Audio flows** through OpenAI Realtime API for speech recognition and response
5. **Your agent's system prompt** guides the AI's behavior
6. **Responses are spoken** back to the caller in real-time
