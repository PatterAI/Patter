## Create Agent

Creates a new AI voice agent with the given configuration.

```bash
curl -X POST https://api.patter.ai/api/v1/agents \
  -H "X-API-Key: pt_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Support Agent",
    "system_prompt": "You are a helpful customer support agent for Acme Corp. Be friendly and concise.",
    "model": "gpt-4o",
    "voice": "alloy",
    "language": "en",
    "first_message": "Hello! Thanks for calling Acme Corp. How can I help you today?",
    "recording_enabled": true
  }'
```

### Request Body

<ParamField body="name" type="string" required>
  Display name for the agent.
</ParamField>

<ParamField body="system_prompt" type="string" required>
  System prompt that defines the agent's behavior. Supports `{variable}` placeholders that are replaced from the `variables` dict at runtime.
</ParamField>

<ParamField body="model" type="string">
  LLM model to use. Defaults to the platform default.
</ParamField>

<ParamField body="voice" type="string">
  TTS voice identifier (e.g., `alloy`, `nova`, `shimmer`).
</ParamField>

<ParamField body="voice_provider" type="string">
  TTS provider (e.g., `openai`, `elevenlabs`).
</ParamField>

<ParamField body="language" type="string">
  Language code (e.g., `en`, `es`, `fr`).
</ParamField>

<ParamField body="first_message" type="string">
  Message the agent speaks when the call begins, after the AI disclosure.
</ParamField>

<ParamField body="tools" type="array">
  List of tool definitions the agent can invoke (function calling).
</ParamField>

<ParamField body="variables" type="object">
  Key-value pairs for dynamic `{placeholder}` replacement in `system_prompt`.
</ParamField>

<ParamField body="guardrails" type="object">
  Guardrail configuration to restrict agent behavior.
</ParamField>

<ParamField body="webhook_url" type="string">
  URL to receive webhook events for this agent's calls.
</ParamField>

<ParamField body="recording_enabled" type="boolean">
  Whether to record calls handled by this agent. Defaults to `false`.
</ParamField>

<ParamField body="client_id" type="string">
  Optional client-side identifier for idempotency or external reference.
</ParamField>

### Response `201`

```json
{
  "data": {
    "id": "agt_xyz789",
    "name": "Support Agent",
    "system_prompt": "You are a helpful customer support agent for Acme Corp...",
    "model": "gpt-4o",
    "voice": "alloy",
    "voice_provider": "openai",
    "language": "en",
    "first_message": "Hello! Thanks for calling Acme Corp. How can I help you today?",
    "tools": [],
    "variables": {},
    "guardrails": null,
    "webhook_url": null,
    "recording_enabled": true,
    "client_id": null,
    "created_at": "2025-01-15T10:30:00Z",
    "updated_at": "2025-01-15T10:30:00Z"
  }
}
```

**Rate limit**: 30 requests per minute.

## List Agents

Returns all agents for the authenticated account.

```bash
curl https://api.patter.ai/api/v1/agents \
  -H "X-API-Key: pt_your_api_key"
```

### Response `200`

```json
{
  "data": [
    {
      "id": "agt_xyz789",
      "name": "Support Agent",
      "model": "gpt-4o",
      "voice": "alloy",
      "language": "en",
      "created_at": "2025-01-15T10:30:00Z",
      "updated_at": "2025-01-15T10:30:00Z"
    }
  ]
}
```

**Rate limit**: 60 requests per minute.

## Get Agent

Returns a single agent by ID.

```bash
curl https://api.patter.ai/api/v1/agents/agt_xyz789 \
  -H "X-API-Key: pt_your_api_key"
```

### Path Parameters

<ParamField path="agent_id" type="string" required>
  The agent ID.
</ParamField>

### Response `200`

Returns the full agent object (same shape as the create response).

## Update Agent

Partially updates an agent. Only include the fields you want to change.

```bash
curl -X PATCH https://api.patter.ai/api/v1/agents/agt_xyz789 \
  -H "X-API-Key: pt_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Premium Support Agent",
    "voice": "nova",
    "first_message": "Welcome to Acme premium support!"
  }'
```

### Path Parameters

<ParamField path="agent_id" type="string" required>
  The agent ID.
</ParamField>

### Request Body

All fields from [Create Agent](#create-agent) are accepted. Only provided fields are updated.

### Response `200`

Returns the updated agent object.

---

## Delete Agent

Permanently deletes an agent.

```bash
curl -X DELETE https://api.patter.ai/api/v1/agents/agt_xyz789 \
  -H "X-API-Key: pt_your_api_key"
```

### Path Parameters

<ParamField path="agent_id" type="string" required>
  The agent ID.
</ParamField>

### Response `204`

No content.
