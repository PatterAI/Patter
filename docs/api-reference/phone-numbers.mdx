## Register Phone Number (BYO)

Register a phone number you already own with a telephony provider (Twilio or Telnyx).

```bash
curl -X POST https://api.patter.ai/api/v1/phone-numbers \
  -H "X-API-Key: pt_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "number": "+14155551234",
    "provider": "twilio",
    "provider_credentials": {
      "account_sid": "AC...",
      "auth_token": "your_auth_token"
    },
    "country": "US"
  }'
```

### Request Body

<ParamField body="number" type="string" required>
  Phone number in E.164 format (e.g., `+14155551234`).
</ParamField>

<ParamField body="provider" type="string" required>
  Telephony provider. One of `twilio` or `telnyx`.
</ParamField>

<ParamField body="provider_credentials" type="object">
  Provider-specific credentials. Encrypted before storage.
</ParamField>

<ParamField body="country" type="string" required>
  ISO 3166-1 alpha-2 country code (e.g., `US`, `GB`, `DE`).
</ParamField>

<ParamField body="stt_config" type="object">
  Speech-to-text configuration overrides for this number.
</ParamField>

<ParamField body="tts_config" type="object">
  Text-to-speech configuration overrides for this number.
</ParamField>

<ParamField body="client_id" type="string">
  Optional client-side identifier for idempotency or external reference.
</ParamField>

### Response `201`

```json
{
  "data": {
    "id": "phn_abc123",
    "number": "+14155551234",
    "provider": "twilio",
    "country": "US",
    "status": "active",
    "agent_id": null,
    "created_at": "2025-01-15T10:30:00Z"
  }
}
```

**Rate limit**: 10 requests per minute.

## Provision Phone Number

Provision a new phone number directly through Patter. Patter purchases the number from the specified provider on your behalf.

```bash
curl -X POST https://api.patter.ai/api/v1/phone-numbers/provision \
  -H "X-API-Key: pt_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "provider": "telnyx",
    "country": "US"
  }'
```

### Request Body

<ParamField body="provider" type="string" required>
  Telephony provider. One of `twilio` or `telnyx`.
</ParamField>

<ParamField body="country" type="string" required>
  ISO 3166-1 alpha-2 country code.
</ParamField>

### Response `201`

```json
{
  "data": {
    "id": "phn_def456",
    "number": "+14155559876",
    "provider": "telnyx",
    "country": "US",
    "status": "active",
    "agent_id": null,
    "created_at": "2025-01-15T10:35:00Z"
  }
}
```

**Rate limit**: 5 requests per minute.

## Assign Agent to Number

Connects an agent to a phone number. Incoming calls to this number will be handled by the assigned agent.

```bash
curl -X POST https://api.patter.ai/api/v1/phone-numbers/phn_abc123/assign-agent \
  -H "X-API-Key: pt_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"agent_id": "agt_xyz789"}'
```

### Path Parameters

<ParamField path="id" type="string" required>
  The phone number ID.
</ParamField>

### Request Body

<ParamField body="agent_id" type="string" required>
  The agent ID to assign.
</ParamField>

### Response `200`

```json
{
  "data": {
    "id": "phn_abc123",
    "number": "+14155551234",
    "provider": "twilio",
    "country": "US",
    "status": "active",
    "agent_id": "agt_xyz789",
    "created_at": "2025-01-15T10:30:00Z"
  }
}
```

**Rate limit**: 30 requests per minute.

## Unassign Agent from Number

Removes the agent assignment from a phone number. Incoming calls will no longer be answered.

```bash
curl -X DELETE https://api.patter.ai/api/v1/phone-numbers/phn_abc123/assign-agent \
  -H "X-API-Key: pt_your_api_key"
```

### Path Parameters

<ParamField path="id" type="string" required>
  The phone number ID.
</ParamField>

### Response `200`

Returns the phone number object with `agent_id` set to `null`.

**Rate limit**: 30 requests per minute.

## Reactivate Phone Number

Reactivates a previously deactivated phone number.

```bash
curl -X POST https://api.patter.ai/api/v1/phone-numbers/phn_abc123/reactivate \
  -H "X-API-Key: pt_your_api_key"
```

### Path Parameters

<ParamField path="id" type="string" required>
  The phone number ID.
</ParamField>

### Response `200`

Returns the phone number object with `status` set to `active`.

**Rate limit**: 10 requests per minute.

## Release Phone Number

Permanently releases a phone number. The number is deregistered from your account and, for provisioned numbers, released back to the provider.

```bash
curl -X DELETE https://api.patter.ai/api/v1/phone-numbers/phn_abc123 \
  -H "X-API-Key: pt_your_api_key"
```

### Path Parameters

<ParamField path="id" type="string" required>
  The phone number ID.
</ParamField>

### Response `204`

No content.

<Warning>
  Releasing a provisioned number returns it to the provider. You may not be able to reclaim the same number.
</Warning>

**Rate limit**: 10 requests per minute.
