# TypeScript SDK

## Installation

```bash
npm install patter
```

## Basic Usage

```typescript
import { Patter } from "patter";

const phone = new Patter({
  mode: "local",
  twilioSid: "ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
  twilioToken: "your_auth_token",
  openaiKey: "sk-xxx",
  phoneNumber: "+15550001234",
  webhookUrl: "abc123.ngrok.io",
});

const agent = phone.agent({
  systemPrompt: "You are a helpful assistant.",
  voice: "alloy",
});

await phone.serve({ agent });
```

## Constructor Options

The `Patter` constructor accepts a `LocalOptions` object for local mode.

<ParamField body="mode" type="'local'" required>
  Must be `"local"` for SDK usage.
</ParamField>

<ParamField body="twilioSid" type="string">
  Twilio Account SID. Required when using Twilio.
</ParamField>

<ParamField body="twilioToken" type="string">
  Twilio Auth Token. Required when `twilioSid` is provided.
</ParamField>

<ParamField body="telnyxKey" type="string">
  Telnyx API key. Required when using Telnyx.
</ParamField>

<ParamField body="telnyxConnectionId" type="string">
  Telnyx Call Control Application ID.
</ParamField>

<ParamField body="telnyxPublicKey" type="string">
  Telnyx Ed25519 public key for webhook signature verification.
</ParamField>

<ParamField body="openaiKey" type="string">
  OpenAI API key for Realtime mode.
</ParamField>

<ParamField body="phoneNumber" type="string" required>
  Your phone number in E.164 format (e.g. `+15550001234`).
</ParamField>

<ParamField body="webhookUrl" type="string" required>
  Public hostname without scheme (e.g. `abc123.ngrok.io`).
</ParamField>

<ParamField body="telephonyProvider" type="'twilio' | 'telnyx'">
  Telephony provider. Auto-detected from `twilioSid` or `telnyxKey`.
</ParamField>

<Note>
Either `twilioSid` or `telnyxKey` must be provided in local mode.
</Note>
