# TypeScript API Reference

## `Patter` Class

### Constructor

```typescript
const phone = new Patter({
  mode: "local",
  twilioSid: "ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
  twilioToken: "your_auth_token",
  openaiKey: "sk-xxx",
  phoneNumber: "+15550001234",
  webhookUrl: "abc123.ngrok.io",
});
```

Accepts either `PatterOptions` (cloud) or `LocalOptions` (local mode). For SDK usage, use `LocalOptions`.

<ParamField body="mode" type="'local'" required>
  Must be `"local"`.
</ParamField>

<ParamField body="twilioSid" type="string">
  Twilio Account SID.
</ParamField>

<ParamField body="twilioToken" type="string">
  Twilio Auth Token. Required when `twilioSid` is set.
</ParamField>

<ParamField body="telnyxKey" type="string">
  Telnyx API key.
</ParamField>

<ParamField body="telnyxConnectionId" type="string">
  Telnyx Call Control Application ID.
</ParamField>

<ParamField body="telnyxPublicKey" type="string">
  Telnyx Ed25519 public key for webhook signature verification.
</ParamField>

<ParamField body="openaiKey" type="string">
  OpenAI API key for Realtime mode.
</ParamField>

<ParamField body="phoneNumber" type="string" required>
  Your phone number in E.164 format.
</ParamField>

<ParamField body="webhookUrl" type="string" required>
  Public hostname without scheme.
</ParamField>

<ParamField body="telephonyProvider" type="'twilio' | 'telnyx'">
  Auto-detected from credentials.
</ParamField>

---

### `phone.agent()`

Creates an `AgentOptions` configuration.

```typescript
phone.agent(opts: AgentOptions): AgentOptions
```

<ParamField body="systemPrompt" type="string" required>
  Instructions for the AI. Supports `{placeholder}` substitution.
</ParamField>

<ParamField body="voice" type="string" default="alloy">
  TTS voice name.
</ParamField>

<ParamField body="model" type="string" default="gpt-4o-mini-realtime-preview">
  OpenAI Realtime model ID.
</ParamField>

<ParamField body="language" type="string" default="en">
  BCP-47 language code.
</ParamField>

<ParamField body="firstMessage" type="string">
  Message spoken immediately when a call connects.
</ParamField>

<ParamField body="provider" type="'openai_realtime' | 'elevenlabs_convai' | 'pipeline'">
  Voice mode. Defaults to `openai_realtime`.
</ParamField>

<ParamField body="tools" type="Array<{ name, description, parameters, webhookUrl }>">
  Tool definitions with webhook URLs.
</ParamField>

<ParamField body="stt" type="STTConfig">
  STT config for pipeline mode.
</ParamField>

<ParamField body="tts" type="TTSConfig">
  TTS config for pipeline mode.
</ParamField>

<ParamField body="elevenlabsKey" type="string">
  ElevenLabs API key (ConvAI mode).
</ParamField>

<ParamField body="elevenlabsAgentId" type="string">
  ElevenLabs agent ID (ConvAI mode).
</ParamField>

<ParamField body="deepgramKey" type="string">
  Deepgram API key (pipeline fallback).
</ParamField>

<ParamField body="variables" type="Record<string, string>">
  Dynamic variable substitution for `systemPrompt`.
</ParamField>

<ParamField body="guardrails" type="Guardrail[]">
  Output guardrails created with `Patter.guardrail()`.
</ParamField>

---

### `phone.serve()`

Starts the embedded server for inbound calls. Returns a `Promise` that resolves when the server stops.

```typescript
phone.serve(opts: ServeOptions): Promise<void>
```

<ParamField body="agent" type="AgentOptions" required>
  Agent configuration.
</ParamField>

<ParamField body="port" type="number" default="8000">
  TCP port (1-65535).
</ParamField>

<ParamField body="recording" type="boolean" default="false">
  Enable call recording.
</ParamField>

<ParamField body="onCallStart" type="(data) => Promise<void>">
  Call start callback.
</ParamField>

<ParamField body="onCallEnd" type="(data) => Promise<void>">
  Call end callback.
</ParamField>

<ParamField body="onTranscript" type="(data) => Promise<void>">
  Per-utterance callback.
</ParamField>

<ParamField body="onMessage" type="(data) => Promise<string>">
  Pipeline mode handler. Return value is spoken.
</ParamField>

<ParamField body="voicemailMessage" type="string">
  AMD voicemail message.
</ParamField>

---

### `phone.call()`

Places an outbound call.

```typescript
phone.call(opts: LocalCallOptions): Promise<void>
```

<ParamField body="to" type="string" required>
  Destination phone number in E.164 format.
</ParamField>

<ParamField body="agent" type="AgentOptions" required>
  Agent configuration.
</ParamField>

<ParamField body="machineDetection" type="boolean" default="false">
  Enable answering machine detection.
</ParamField>

<ParamField body="voicemailMessage" type="string">
  Voicemail message for AMD.
</ParamField>

<ParamField body="variables" type="Record<string, string>">
  Per-call variable overrides.
</ParamField>

---

```typescript
phone.disconnect(): Promise<void>

## Static Methods

```

```typescript

```

### `Patter.elevenlabs()`

```typescript
Patter.elevenlabs(opts: { apiKey: string; voice?: string }): TTSConfig
```

Default voice: `"rachel"`.

### `Patter.openaiTts()`

```typescript
Patter.openaiTts(opts: { apiKey: string; voice?: string }): TTSConfig
```

Default voice: `"alloy"`.

### `Patter.guardrail()`

```typescript
Patter.guardrail(opts: {
  name: string;
  blockedTerms?: string[];
  check?: (text: string) => boolean;
  replacement?: string;

```
