# Calls

## Inbound Calls — `serve()`

The `serve()` method starts an embedded HTTP + WebSocket server that handles incoming calls from your telephony provider.

```typescript
import { Patter } from "patter";

const phone = new Patter({
  mode: "local",
  twilioSid: "ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
  twilioToken: "your_auth_token",
  openaiKey: "sk-xxx",
  phoneNumber: "+15550001234",
  webhookUrl: "abc123.ngrok.io",
});

const agent = phone.agent({
  systemPrompt: "You are a helpful assistant.",
  voice: "alloy",
});

await phone.serve({ agent, port: 8000 });
```

### `serve()` Options

<ParamField body="agent" type="AgentOptions" required>
  The agent configuration for all incoming calls. Created via `phone.agent()`.
</ParamField>

<ParamField body="port" type="number" default="8000">
  TCP port to bind the server to (1-65535).
</ParamField>

<ParamField body="recording" type="boolean" default="false">
  When `true`, record each call via the telephony provider's recording API.
</ParamField>

<ParamField body="onCallStart" type="(data: Record<string, unknown>) => Promise<void>">
  Async callback fired when a call starts.
</ParamField>

<ParamField body="onCallEnd" type="(data: Record<string, unknown>) => Promise<void>">
  Async callback fired when a call ends.
</ParamField>

<ParamField body="onTranscript" type="(data: Record<string, unknown>) => Promise<void>">
  Async callback fired for each utterance.
</ParamField>

<ParamField body="onMessage" type="(data: Record<string, unknown>) => Promise<string>">
  Pipeline mode only. Called with the user's transcript; return value is spoken to the caller.
</ParamField>

<ParamField body="voicemailMessage" type="string">
  Message spoken when AMD detects an answering machine.
</ParamField>

## Outbound Calls — `call()`

Place an outbound call to any phone number.

```typescript
await phone.call({
  to: "+15550005678",
  agent,
});
```

<Warning>
The embedded server must be running before placing an outbound call. The telephony provider connects back to your server to stream audio.
</Warning>

### `call()` Options

<ParamField body="to" type="string" required>
  Phone number to call in E.164 format (e.g. `+15550005678`).
</ParamField>

<ParamField body="agent" type="AgentOptions" required>
  The agent configuration to use for this call.
</ParamField>

<ParamField body="machineDetection" type="boolean" default="false">
  Enable answering machine detection.
</ParamField>

<ParamField body="voicemailMessage" type="string">
  Message spoken when AMD detects a machine. Requires `machineDetection: true`.
</ParamField>

<ParamField body="variables" type="Record<string, string>">
  Dynamic variables merged into `agent.variables` before the call. Overrides agent-level values.
</ParamField>

## Callbacks

All callbacks receive a `Record<string, unknown>` with contextual data.

### `onCallStart`

```typescript
await phone.serve({
  agent,
  onCallStart: async (data) => {
    console.log(`Call started: ${data.call_id}`);
    console.log(`Direction: ${data.direction}`);
    console.log(`Caller: ${data.caller}`);
  },
});
```

### `onCallEnd`

```typescript
await phone.serve({
  agent,
  onCallEnd: async (data) => {
    console.log(`Call ended: ${data.call_id}`);
    console.log(`Duration: ${data.duration}s`);
    console.log("History:", data.history);
  },
});
```

### `onTranscript`

```typescript
await phone.serve({
  agent,
  onTranscript: async (data) => {
    console.log(`[${data.role}]: ${data.text}`);
  },
});
```

<Note>
The `data.history` field in callbacks contains the full conversation as `[{role, text, timestamp}]`.
</Note>
