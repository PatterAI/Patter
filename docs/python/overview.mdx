# Python SDK

## Installation

```bash
pip install patter[local]
```

The `[local]` extra installs dependencies for running the embedded server (FastAPI, uvicorn, WebSocket support).

## Basic Usage

```python
import asyncio
from patter import Patter

phone = Patter(
    twilio_sid="ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
    twilio_token="your_auth_token",
    openai_key="sk-xxx",
    phone_number="+15550001234",
    webhook_url="abc123.ngrok.io",
)

agent = phone.agent(
    system_prompt="You are a helpful assistant.",
    voice="alloy",
)

asyncio.run(phone.serve(agent))
```

## Constructor Parameters

<ParamField body="twilio_sid" type="str" default="">
  Twilio Account SID. Required when using Twilio as telephony provider.
</ParamField>

<ParamField body="twilio_token" type="str" default="">
  Twilio Auth Token. Required when `twilio_sid` is provided.
</ParamField>

<ParamField body="telnyx_key" type="str" default="">
  Telnyx API key. Required when using Telnyx as telephony provider.
</ParamField>

<ParamField body="telnyx_connection_id" type="str" default="">
  Telnyx Call Control Application ID. Used with Telnyx telephony.
</ParamField>

<ParamField body="openai_key" type="str" default="">
  OpenAI API key. Required for `openai_realtime` voice mode.
</ParamField>

<ParamField body="elevenlabs_key" type="str" default="">
  ElevenLabs API key. Used as fallback TTS in pipeline mode when `tts` is not specified on the agent.
</ParamField>

<ParamField body="deepgram_key" type="str" default="">
  Deepgram API key. Used as fallback STT in pipeline mode when `stt` is not specified on the agent.
</ParamField>

<ParamField body="phone_number" type="str" default="">
  Your phone number in E.164 format (e.g. `+15550001234`). Required in local mode.
</ParamField>

<ParamField body="webhook_url" type="str" default="">
  Public hostname of this server without the scheme (e.g. `abc123.ngrok.io`). Required in local mode.
</ParamField>

<Note>
Mode is auto-detected: if you pass `twilio_sid` or `telnyx_key` without an `api_key`, the SDK enters local mode automatically.
</Note>
