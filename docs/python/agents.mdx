# Agents

The `phone.agent()` method creates an `Agent` configuration that defines how the AI behaves on calls.

## Basic Agent

```python
agent = phone.agent(
    system_prompt="You are a friendly receptionist for Acme Corp.",
    voice="alloy",
)
```

## Full Configuration

```python
agent = phone.agent(
    system_prompt="You are {name}'s assistant. Today is {date}.",
    voice="echo",
    model="gpt-4o-mini-realtime-preview",
    language="en",
    first_message="Hello! How can I help you today?",
    provider="openai_realtime",
    tools=[
        {
            "name": "check_order",
            "description": "Look up an order by ID",
            "parameters": {
                "type": "object",
                "properties": {
                    "order_id": {"type": "string", "description": "The order ID"}
                },
                "required": ["order_id"],
            },
            "webhook_url": "https://api.example.com/orders/lookup",
        }
    ],
    variables={"name": "Alice", "date": "2026-04-03"},
    guardrails=[
        Patter.guardrail(
            name="no_medical",
            blocked_terms=["diagnosis", "prescription"],
            replacement="I'm not qualified to give medical advice.",
        ),
    ],
)
```

## Parameters

<ParamField body="system_prompt" type="str" required>
  Instructions for the AI agent. Supports `{placeholder}` syntax for dynamic variable substitution.
</ParamField>

<ParamField body="voice" type="str" default="alloy">
  TTS voice name. For OpenAI Realtime: `alloy`, `echo`, `fable`, `onyx`, `nova`, `shimmer`.
</ParamField>

<ParamField body="model" type="str" default="gpt-4o-mini-realtime-preview">
  OpenAI Realtime model ID.
</ParamField>

<ParamField body="language" type="str" default="en">
  BCP-47 language code (e.g. `en`, `es`, `fr`, `it`).
</ParamField>

<ParamField body="first_message" type="str" default="">
  If set, the agent speaks this message immediately when a call connects.
</ParamField>

<ParamField body="provider" type="str" default="openai_realtime">
  Voice mode: `openai_realtime`, `elevenlabs_convai`, or `pipeline`.
</ParamField>

<ParamField body="tools" type="list[dict] | None" default="None">
  List of tool definitions. Each tool must have `name`, `description`, `parameters`, and `webhook_url`. See [Tool Calling](/examples/tool-calling) for details.
</ParamField>

<ParamField body="stt" type="STTConfig | None" default="None">
  STT provider for pipeline mode. Use `Patter.deepgram()` or `Patter.whisper()`.
</ParamField>

<ParamField body="tts" type="TTSConfig | None" default="None">
  TTS provider for pipeline mode. Use `Patter.elevenlabs()` or `Patter.openai_tts()`.
</ParamField>

<ParamField body="variables" type="dict | None" default="None">
  Key-value pairs substituted into `system_prompt` at call time. `{name}` in the prompt is replaced with the value of `variables["name"]`.
</ParamField>

<ParamField body="guardrails" type="list | None" default="None">
  List of guardrail dicts created with `Patter.guardrail()`. See [Guardrails](/examples/guardrails).
</ParamField>
