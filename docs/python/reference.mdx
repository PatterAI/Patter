# Python API Reference

## `Patter` Class

The main SDK client. Initializing with telephony credentials (without `api_key`) enters local mode.

### Constructor

```python
phone = Patter(
    twilio_sid="ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
    twilio_token="your_auth_token",
    openai_key="sk-xxx",
    phone_number="+15550001234",
    webhook_url="abc123.ngrok.io",
)
```

<ParamField body="twilio_sid" type="str" default="">
  Twilio Account SID.
</ParamField>

<ParamField body="twilio_token" type="str" default="">
  Twilio Auth Token. Required when `twilio_sid` is set.
</ParamField>

<ParamField body="telnyx_key" type="str" default="">
  Telnyx API key.
</ParamField>

<ParamField body="telnyx_connection_id" type="str" default="">
  Telnyx Call Control Application ID.
</ParamField>

<ParamField body="openai_key" type="str" default="">
  OpenAI API key for Realtime mode.
</ParamField>

<ParamField body="elevenlabs_key" type="str" default="">
  ElevenLabs API key (pipeline mode fallback).
</ParamField>

<ParamField body="deepgram_key" type="str" default="">
  Deepgram API key (pipeline mode fallback).
</ParamField>

<ParamField body="phone_number" type="str" default="">
  Your phone number in E.164 format.
</ParamField>

<ParamField body="webhook_url" type="str" default="">
  Public hostname without scheme (e.g. `abc123.ngrok.io`).
</ParamField>

### `phone.agent()`

Creates an `Agent` configuration.

```python
agent = phone.agent(
    system_prompt: str,
    voice: str = "alloy",
    model: str = "gpt-4o-mini-realtime-preview",
    language: str = "en",
    first_message: str = "",
    tools: list[dict] | None = None,
    provider: str = "openai_realtime",
    stt: STTConfig | None = None,
    tts: TTSConfig | None = None,
    variables: dict | None = None,
    guardrails: list | None = None,
) -> Agent
```

<ParamField body="system_prompt" type="str" required>
  Instructions for the AI. Supports `{placeholder}` substitution via `variables`.
</ParamField>

<ParamField body="voice" type="str" default="alloy">
  TTS voice name.
</ParamField>

<ParamField body="model" type="str" default="gpt-4o-mini-realtime-preview">
  OpenAI Realtime model ID.
</ParamField>

<ParamField body="language" type="str" default="en">
  BCP-47 language code.
</ParamField>

<ParamField body="first_message" type="str" default="">
  Message spoken immediately when a call connects.
</ParamField>

<ParamField body="tools" type="list[dict] | None" default="None">
  Tool definitions. Each dict must have `name` (str), `description` (str), `parameters` (dict), and `webhook_url` (str).
</ParamField>

<ParamField body="provider" type="str" default="openai_realtime">
  Voice mode: `openai_realtime`, `elevenlabs_convai`, or `pipeline`.
</ParamField>

<ParamField body="stt" type="STTConfig | None" default="None">
  STT config for pipeline mode.
</ParamField>

<ParamField body="tts" type="TTSConfig | None" default="None">
  TTS config for pipeline mode.
</ParamField>

<ParamField body="variables" type="dict | None" default="None">
  Key-value pairs for `{placeholder}` substitution in `system_prompt`.
</ParamField>

<ParamField body="guardrails" type="list | None" default="None">
  Output guardrails created with `Patter.guardrail()`.
</ParamField>

### `await phone.serve()`

Starts the embedded server for inbound calls. Blocks until stopped.

```python
await phone.serve(
    agent: Agent,
    port: int = 8000,
    recording: bool = False,
    on_call_start: Callable | None = None,
    on_call_end: Callable | None = None,
    on_transcript: Callable | None = None,
    on_message: Callable | None = None,
    voicemail_message: str = "",
) -> None
```

<ParamField body="agent" type="Agent" required>
  Agent configuration created via `phone.agent()`.
</ParamField>

<ParamField body="port" type="int" default="8000">
  TCP port to bind (1-65535).
</ParamField>

<ParamField body="recording" type="bool" default="False">
  Enable call recording.
</ParamField>

<ParamField body="on_call_start" type="Callable[[dict], Awaitable[None]] | None" default="None">
  Async callback for call start events.
</ParamField>

<ParamField body="on_call_end" type="Callable[[dict], Awaitable[None]] | None" default="None">
  Async callback for call end events.
</ParamField>

<ParamField body="on_transcript" type="Callable[[dict], Awaitable[None]] | None" default="None">
  Async callback for each transcript utterance.
</ParamField>

<ParamField body="on_message" type="Callable[[dict], Awaitable[str]] | None" default="None">
  Pipeline mode message handler. Return value is spoken to the caller.
</ParamField>

<ParamField body="voicemail_message" type="str" default="">
  Voicemail message for AMD.
</ParamField>

---

### `await phone.call()`

Places an outbound call.

```python
await phone.call(
    to: str,
    agent: Agent,
    from_number: str = "",
    machine_detection: bool = False,
    voicemail_message: str = "",
) -> None
```

<ParamField body="to" type="str" required>
  Destination phone number in E.164 format.
</ParamField>

<ParamField body="agent" type="Agent" required>
  Agent configuration for the call.
</ParamField>

<ParamField body="from_number" type="str" default="">
  Caller ID. Defaults to `phone_number` from constructor.
</ParamField>

<ParamField body="machine_detection" type="bool" default="False">
  Enable answering machine detection.
</ParamField>

<ParamField body="voicemail_message" type="str" default="">
  Message spoken when AMD detects a machine.
</ParamField>

---

### `await phone.disconnect()`

Stops the embedded server and cleans up resources.

```python
await phone.disconnect()
```

## Data Classes

### `Agent`

Frozen dataclass returned by `phone.agent()`.

| Field | Type | Default |
|-------|------|---------|
| `system_prompt` | `str` | (required) |
| `voice` | `str` | `"alloy"` |
| `model` | `str` | `"gpt-4o-mini-realtime-preview"` |
| `language` | `str` | `"en"` |
| `first_message` | `str` | `""` |
| `tools` | `list[dict] \| None` | `None` |
| `provider` | `str` | `"openai_realtime"` |
| `stt` | `STTConfig \| None` | `None` |
| `tts` | `TTSConfig \| None` | `None` |
| `variables` | `dict \| None` | `None` |
| `guardrails` | `list \| None` | `None` |

### `STTConfig`

Frozen dataclass for speech-to-text configuration.

| Field | Type | Default |
|-------|------|---------|
| `provider` | `str` | (required) |
| `api_key` | `str` | (required) |
| `language` | `str` | `"en"` |

### `TTSConfig`

Frozen dataclass for text-to-speech configuration.

| Field | Type | Default |
|-------|------|---------|
| `provider` | `str` | (required) |
| `api_key` | `str` | (required) |
| `voice` | `str` | `"alloy"` |

### `IncomingMessage`

Frozen dataclass for incoming messages.

| Field | Type |
|-------|------|
| `text` | `str` |
| `call_id` | `str` |
| `caller` | `str` |
