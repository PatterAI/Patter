# Calls

## Inbound Calls — `serve()`

The `serve()` method starts an embedded HTTP + WebSocket server that handles incoming calls from your telephony provider.

```python
import asyncio
from patter import Patter

phone = Patter(
    twilio_sid="ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
    twilio_token="your_auth_token",
    openai_key="sk-xxx",
    phone_number="+15550001234",
    webhook_url="abc123.ngrok.io",
)

agent = phone.agent(
    system_prompt="You are a helpful assistant.",
    voice="alloy",
)

asyncio.run(phone.serve(agent, port=8000))
```

### `serve()` Parameters

<ParamField body="agent" type="Agent" required>
  The agent configuration to use for all incoming calls. Created via `phone.agent()`.
</ParamField>

<ParamField body="port" type="int" default="8000">
  TCP port to bind the server to.
</ParamField>

<ParamField body="recording" type="bool" default="False">
  When `True`, record each call via the telephony provider's recording API.
</ParamField>

<ParamField body="on_call_start" type="Callable[[dict], Awaitable[None]] | None" default="None">
  Async callback fired when a call starts. Receives a dict with call metadata.
</ParamField>

<ParamField body="on_call_end" type="Callable[[dict], Awaitable[None]] | None" default="None">
  Async callback fired when a call ends. Receives a dict with call metadata including duration.
</ParamField>

<ParamField body="on_transcript" type="Callable[[dict], Awaitable[None]] | None" default="None">
  Async callback fired for each utterance in the conversation.
</ParamField>

<ParamField body="on_message" type="Callable[[dict], Awaitable[str]] | None" default="None">
  Pipeline mode only. Called with the user's transcribed text; the return value is synthesized to speech and played back.
</ParamField>

<ParamField body="voicemail_message" type="str" default="">
  Message spoken when AMD detects an answering machine. Requires `machine_detection=True` on the outbound `call()`.
</ParamField>

## Outbound Calls — `call()`

Place an outbound call to any phone number.

```python
await phone.call(
    to="+15550005678",
    agent=agent,
)
```

<Warning>
The embedded server must be running before placing an outbound call. The telephony provider connects back to your server to stream audio.
</Warning>

### `call()` Parameters

<ParamField body="to" type="str" required>
  Phone number to call in E.164 format (e.g. `+15550005678`).
</ParamField>

<ParamField body="agent" type="Agent" required>
  The agent configuration to use for this call.
</ParamField>

<ParamField body="from_number" type="str" default="">
  Number to call from. If empty, uses the `phone_number` from the constructor.
</ParamField>

<ParamField body="machine_detection" type="bool" default="False">
  Enable answering machine detection. When a machine is detected, the `voicemail_message` (if set on `serve()`) is played and the call ends.
</ParamField>

<ParamField body="voicemail_message" type="str" default="">
  Message spoken when AMD detects a machine. Requires `machine_detection=True`.
</ParamField>

## Callbacks

All callbacks receive a `dict` with contextual data.

### `on_call_start`

```python
async def on_call_start(data):
    print(f"Call started: {data['call_id']}")
    print(f"Direction: {data['direction']}")
    print(f"Caller: {data['caller']}")

await phone.serve(agent, on_call_start=on_call_start)
```

### `on_call_end`

```python
async def on_call_end(data):
    print(f"Call ended: {data['call_id']}")
    print(f"Duration: {data['duration']}s")
    print(f"History: {data['history']}")

await phone.serve(agent, on_call_end=on_call_end)
```

### `on_transcript`

```python
async def on_transcript(data):
    print(f"[{data['role']}]: {data['text']}")

await phone.serve(agent, on_transcript=on_transcript)
```

<Note>
The `data.history` field in callbacks contains the full conversation as `[{"role": "...", "text": "...", "timestamp": "..."}]`.
</Note>
