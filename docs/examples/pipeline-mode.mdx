# Pipeline Mode Example

Pipeline mode gives you full control over each step of the voice pipeline: speech-to-text, language model processing, and text-to-speech. Use it when you need a non-OpenAI LLM or want to mix providers.

<CodeGroup>
```python Python
import asyncio
from patter import Patter

phone = Patter(
    twilio_sid="ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
    twilio_token="your_auth_token",
    phone_number="+15550001234",
    webhook_url="abc123.ngrok.io",
)

agent = phone.agent(
    system_prompt="You are a helpful assistant.",
    provider="pipeline",
    stt=Patter.deepgram(api_key="your_deepgram_key"),
    tts=Patter.elevenlabs(api_key="your_elevenlabs_key", voice="rachel"),
)


async def handle_message(data):
    user_text = data["text"]
    history = data.get("history", [])

    # Use any LLM here — Claude, Mistral, Llama, etc.
    # This example uses a simple echo for demonstration.
    return f"You said: {user_text}"


asyncio.run(
    phone.serve(agent, on_message=handle_message)
)
```

```typescript TypeScript
import { Patter } from "patter";

const phone = new Patter({
  mode: "local",
  twilioSid: "ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
  twilioToken: "your_auth_token",
  phoneNumber: "+15550001234",
  webhookUrl: "abc123.ngrok.io",
});

const agent = phone.agent({
  systemPrompt: "You are a helpful assistant.",
  provider: "pipeline",
  stt: Patter.deepgram({ apiKey: "your_deepgram_key" }),
  tts: Patter.elevenlabs({ apiKey: "your_elevenlabs_key", voice: "rachel" }),
});

await phone.serve({
  agent,
  onMessage: async (data) => {
    const userText = data.text as string;
    const history = data.history as Array<{ role: string; text: string }>;

    // Use any LLM here — Claude, Mistral, Llama, etc.
    return `You said: ${userText}`;
  },
});
```
</CodeGroup>

## How It Works

In pipeline mode, the audio flow is:

1. **Caller speaks** -- audio streams from the telephony provider
2. **STT** (Deepgram or Whisper) transcribes the audio to text
3. **`on_message` callback** receives the text and conversation history. You call your own LLM and return the response as a string.
4. **TTS** (ElevenLabs or OpenAI TTS) synthesizes the response to audio
5. **Audio plays** back to the caller

## Using with Claude

<CodeGroup>
```python Python
import anthropic

client = anthropic.Anthropic(api_key="your_anthropic_key")

async def handle_message(data):
    user_text = data["text"]
    history = data.get("history", [])

    messages = [{"role": e["role"], "content": e["text"]} for e in history]
    messages.append({"role": "user", "content": user_text})

    response = client.messages.create(
        model="claude-sonnet-4-20250514",
        max_tokens=200,
        system="You are a helpful phone assistant. Keep responses under 2 sentences.",
        messages=messages,
    )
    return response.content[0].text
```

```typescript TypeScript
import Anthropic from "@anthropic-ai/sdk";

const anthropic = new Anthropic({ apiKey: "your_anthropic_key" });

async function handleMessage(data: Record<string, unknown>): Promise<string> {
  const userText = data.text as string;
  const history = (data.history ?? []) as Array<{ role: string; text: string }>;

  const messages = history.map((e) => ({
    role: e.role as "user" | "assistant",
    content: e.text,
  }));
  messages.push({ role: "user", content: userText });

  const response = await anthropic.messages.create({
    model: "claude-sonnet-4-20250514",
    max_tokens: 200,
    system: "You are a helpful phone assistant. Keep responses under 2 sentences.",
    messages,
  });

  return (response.content[0] as { text: string }).text;
}
```
</CodeGroup>

<Tip>
Keep LLM responses short (1-2 sentences) for a natural phone conversation experience. Long responses create awkward pauses.
</Tip>

## Provider Combinations

| STT | TTS | Use Case |
|-----|-----|----------|
| Deepgram | ElevenLabs | Best quality voice, fast transcription |
| Deepgram | OpenAI TTS | Lower cost, good quality |
| Whisper | ElevenLabs | Best transcription accuracy, best voice |
| Whisper | OpenAI TTS | Simplest setup (all OpenAI) |
