# Recording & AMD Example

This example demonstrates call recording and answering machine detection (AMD) with automatic voicemail drop.

## Call Recording

Enable recording with the `recording` flag on `serve()`. Recordings are stored via your telephony provider.

<CodeGroup>
```python Python
import asyncio
from patter import Patter

phone = Patter(
    twilio_sid="ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
    twilio_token="your_auth_token",
    openai_key="sk-xxx",
    phone_number="+15550001234",
    webhook_url="abc123.ngrok.io",
)

agent = phone.agent(
    system_prompt="You are a helpful assistant.",
    voice="alloy",
)


async def on_call_end(data):
    print(f"Call recorded — ID: {data.get('call_id')}")
    print(f"Duration: {data.get('duration', 0)}s")


asyncio.run(
    phone.serve(
        agent,
        recording=True,
        on_call_end=on_call_end,
    )
)
```

```typescript TypeScript
import { Patter } from "patter";

const phone = new Patter({
  mode: "local",
  twilioSid: "ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
  twilioToken: "your_auth_token",
  openaiKey: "sk-xxx",
  phoneNumber: "+15550001234",
  webhookUrl: "abc123.ngrok.io",
});

const agent = phone.agent({
  systemPrompt: "You are a helpful assistant.",
  voice: "alloy",
});

await phone.serve({
  agent,
  recording: true,
  onCallEnd: async (data) => {
    console.log(`Call recorded — ID: ${data.call_id}`);
    console.log(`Duration: ${data.duration ?? 0}s`);
  },
});
```
</CodeGroup>

## Answering Machine Detection (AMD)

AMD detects whether an outbound call was answered by a human or a machine. When a machine is detected, you can automatically leave a voicemail message.

<CodeGroup>
```python Python
import asyncio
from patter import Patter

phone = Patter(
    twilio_sid="ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
    twilio_token="your_auth_token",
    openai_key="sk-xxx",
    phone_number="+15550001234",
    webhook_url="abc123.ngrok.io",
)

agent = phone.agent(
    system_prompt="You are calling to confirm an appointment.",
    voice="alloy",
    first_message="Hi! I'm calling from Acme Corp about your appointment.",
)


async def main():
    serve_task = asyncio.create_task(
        phone.serve(
            agent,
            voicemail_message="Hi, this is Acme Corp calling about your appointment tomorrow at 3 PM. Please call us back at 555-000-1234 to confirm. Thank you!",
        )
    )

    await asyncio.sleep(2)

    # Place call with AMD enabled
    await phone.call(
        to="+15550005678",
        agent=agent,
        machine_detection=True,
        voicemail_message="Hi, this is Acme Corp calling about your appointment tomorrow at 3 PM. Please call us back at 555-000-1234 to confirm. Thank you!",
    )

    await serve_task


asyncio.run(main())
```

```typescript TypeScript
import { Patter } from "patter";

const phone = new Patter({
  mode: "local",
  twilioSid: "ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
  twilioToken: "your_auth_token",
  openaiKey: "sk-xxx",
  phoneNumber: "+15550001234",
  webhookUrl: "abc123.ngrok.io",
});

const agent = phone.agent({
  systemPrompt: "You are calling to confirm an appointment.",
  voice: "alloy",
  firstMessage: "Hi! I'm calling from Acme Corp about your appointment.",
});

const servePromise = phone.serve({
  agent,
  voicemailMessage:
    "Hi, this is Acme Corp calling about your appointment tomorrow at 3 PM. Please call us back at 555-000-1234 to confirm. Thank you!",
});

setTimeout(async () => {
  await phone.call({
    to: "+15550005678",
    agent,
    machineDetection: true,
    voicemailMessage:
      "Hi, this is Acme Corp calling about your appointment tomorrow at 3 PM. Please call us back at 555-000-1234 to confirm. Thank you!",
  });
}, 2000);

await servePromise;
```
</CodeGroup>

## How AMD Works

1. You enable `machine_detection=True` / `machineDetection: true` on the `call()` method
2. The telephony provider analyzes the first few seconds of audio
3. If a human answers, the agent proceeds with the normal conversation
4. If an answering machine is detected:
   - The `voicemail_message` is spoken
   - The call is automatically ended after the message plays

<Note>
AMD uses Twilio's `DetectMessageEnd` mode, which waits for the voicemail greeting to finish before playing the message. This ensures the voicemail is recorded properly.
</Note>

## Combined Example

Recording and AMD work together. This example records all calls and drops a voicemail on machines:

<CodeGroup>
```python Python
asyncio.run(
    phone.serve(
        agent,
        recording=True,
        voicemail_message="Please call us back at 555-000-1234.",
    )
)
```

```typescript TypeScript
await phone.serve({
  agent,
  recording: true,
  voicemailMessage: "Please call us back at 555-000-1234.",
});
```
