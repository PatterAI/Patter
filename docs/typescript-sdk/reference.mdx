# API Reference

Complete reference for all types, interfaces, and exports in the `patter` package.

## Patter Class

### Constructor

```typescript
new Patter(options: PatterOptions | LocalOptions)
```

### Instance Methods

| Method | Signature | Description |
|--------|-----------|-------------|
| `agent` | `(opts: AgentOptions) => AgentOptions` | Validates and returns an agent configuration. |
| `serve` | `(opts: ServeOptions) => Promise<void>` | Starts the embedded server (local mode only). |
| `connect` | `(opts: ConnectOptions) => Promise<void>` | Connects to the Patter cloud platform. |
| `call` | `(opts: CallOptions \| LocalCallOptions) => Promise<void>` | Makes an outbound call. |
| `disconnect` | `() => Promise<void>` | Disconnects from the cloud platform. |
| `createAgent` | `(opts: CreateAgentOptions) => Promise<Agent>` | Creates a cloud-managed agent. |
| `listAgents` | `() => Promise<Agent[]>` | Lists all cloud-managed agents. |
| `buyNumber` | `(opts?: { country?: string; provider?: string }) => Promise<PhoneNumber>` | Provisions a new phone number. |
| `assignAgent` | `(numberId: string, agentId: string) => Promise<void>` | Assigns an agent to a phone number. |
| `listCalls` | `(limit?: number) => Promise<Call[]>` | Lists recent calls (1-1000, default 50). |
| `test` | `(agent: AgentOptions, opts?: TestOptions) => Promise<void>` | Starts an interactive terminal test session (local mode only). See [Test Mode](/typescript-sdk/test-mode). |

### Static Methods

| Method | Signature | Description |
|--------|-----------|-------------|
| `Patter.deepgram` | `(opts: { apiKey: string; language?: string }) => STTConfig` | Creates a Deepgram STT config. |
| `Patter.whisper` | `(opts: { apiKey: string; language?: string }) => STTConfig` | Creates a Whisper STT config. |
| `Patter.elevenlabs` | `(opts: { apiKey: string; voice?: string }) => TTSConfig` | Creates an ElevenLabs TTS config. |
| `Patter.openaiTts` | `(opts: { apiKey: string; voice?: string }) => TTSConfig` | Creates an OpenAI TTS config. |
| `Patter.guardrail` | `(opts: { name: string; blockedTerms?: string[]; check?: (text: string) => boolean; replacement?: string }) => Guardrail` | Creates a guardrail config. |
| `Patter.tool` | `(opts: { name: string; description?: string; parameters?: object; handler?: Function; webhookUrl?: string }) => ToolDefinition` | Creates a tool definition. Requires either `handler` or `webhookUrl`. |

## Error Classes

All errors extend `PatterError`, which extends the native `Error` class.

| Class | Description |
|-------|-------------|
| `PatterError` | Base error class for all SDK errors. |
| `PatterConnectionError` | WebSocket connection failures or disconnection errors. |
| `AuthenticationError` | Invalid API key or authentication failure. |
| `ProvisionError` | Failed to provision resources (create agents, buy numbers, make calls). |

```typescript
import {
  PatterError,
  PatterConnectionError,
  AuthenticationError,
  ProvisionError,
} from "getpatter";

try {
  await phone.call({ to: "+15559876543", agent });
} catch (error) {
  if (error instanceof ProvisionError) {
    console.error("Failed to make call:", error.message);
  } else if (error instanceof PatterConnectionError) {
    console.error("Connection issue:", error.message);
  }
}
```
