# What is Patter?

Patter is an open-source SDK that gives your AI agent a voice and a phone number. Write a message handler, and Patter takes care of telephony, speech-to-text, text-to-speech, and WebSocket streaming.

## How it works

```
Phone Call → Telephony (Twilio/Telnyx) → WebSocket → Patter → STT → Your Agent → TTS → Caller
```

Your agent receives transcribed speech as text and returns a text response. Patter handles everything else — audio streaming, transcoding, provider APIs, and real-time voice synthesis.

## Deployment modes

| Mode | Description | Best for |
|------|-------------|----------|
| **Local** | Fully embedded server, no Patter backend needed | Development, self-hosted production |
| **Cloud** | Managed infrastructure via Patter API | Production with minimal ops |
| **Self-hosted** | Your own Patter backend deployment | Full control with cloud agent management |
