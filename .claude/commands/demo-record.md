# Record Demo

Guide for recording the 1-minute demo videos for hackathon submission.

## Pre-Recording Checklist
- [ ] Both servers running (ShipCall on 8080, VoiceScope on 8080)
- [ ] ngrok tunnel active and stable
- [ ] Test call works (run /test-call first)
- [ ] Screen recording software ready (QuickTime or OBS)
- [ ] Phone on speaker mode, volume up
- [ ] Quiet environment (use headphones if recording voiceover separately)

## ShipCall Demo Script (60 seconds)
1. [0:00-0:10] Show the FastShot mobile app — event feed with recent code events
2. [0:10-0:15] Tap "Call Me Now" button
3. [0:15-0:20] Phone rings — answer on speaker
4. [0:20-0:40] AI explains the error spike, reads logs, suggests a fix
5. [0:40-0:50] Developer asks a follow-up question, AI responds
6. [0:50-0:60] End card: "ShipCall — Your codebase calls you. Built on Patter."

## VoiceScope Demo Script (60 seconds)
1. [0:00-0:10] Show the web UI — upload a photo of a circuit board
2. [0:10-0:15] Enter phone number, click "Analyze & Call Me"
3. [0:15-0:25] Status shows "Analyzing..." then "Calling you now..."
4. [0:25-0:50] Phone rings — AI expert explains what it sees, identifies the loose cable, gives fix instructions
5. [0:50-0:60] End card: "VoiceScope — See anything. Hear an expert. Built on Patter."

## Upload
1. Upload both videos to YouTube as UNLISTED
2. At submission time (2:30 PM), switch to PUBLIC
3. Add descriptions with hashtags: #StanfordHackathon #DeepMind #GeminiAI #Patter #VoiceAI
