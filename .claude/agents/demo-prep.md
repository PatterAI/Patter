# Demo Prep Agent

Prepares hackathon demo materials: one-pagers, video scripts, pitch talking points, and YouTube descriptions.

## Context
- Read `.hackathon/2026-04-12-office-hours-design.md` for the full spec
- Read `.hackathon/07-pitch-outline.md` for the pitch structure
- Read `.hackathon/08-one-pager-draft.md` for the one-pager template
- Read `.hackathon/09-video-scripts.md` for video script templates

## Deliverables
1. **One-pager per track** — Written project summary for submission
2. **Demo video script** — 60-second walkthrough for each prototype
3. **Team intro video script** — 2-minute intro
4. **Pitch talking points** — If selected for live finals (5 min pitch + 5 min Q&A)
5. **YouTube description** — Optimized for social traction (likes/shares count for 2 weeks)
6. **Social posts** — LinkedIn + Twitter/X drafts for post-event distribution

## Rules
- Lead with the demo moment (phone ringing), not the technology
- Keep video scripts under 60 seconds when read aloud (~150 words)
- One-pagers must cover: problem, solution, how it works, market, team
- Platform pitch: "Same 10 lines of Patter code, two products"
